# Twitter4jTutorial

Twitter4jのチュートリアルプログラム

## Description

全公開ツイートの1%を抽出する

CSVファイルも作成可能

## Requirement

twitter4j: http://twitter4j.org/ja/index.html

## Usage

srcフォルダ直下にtwitter4j.propertiesを作成

    debug=true
    oauth.consumerKey=****************
    oauth.consumerSecret=****************
    oauth.accessToken=****************
    oauth.accessTokenSecret=****************

twitter4jライブラリを入れてプログラムを起動
